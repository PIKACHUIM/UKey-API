======================================================
    目录 & 说明
======================================================

ngp11v211.dll ngp11v211.lib
ePassNG中间件中PKCS#11接口的动态链接库和LIB文件，基于PKCS#11 v2.11规
范编写。

------------------------------------------------------
ImportCertsDll.dll 和 ImportCertsDll.lib
ePassNG中间件导入证书的库文件；

------------------------------------------------------
init_fteps3k_sc.dll init_fteps3k_sc.lib
初始化ePass3000的动态链接库和LIB文件，可以设置公私有区大小和PIN码重试次数。

------------------------------------------------------
msvcp60.dll : 标准的C运行库程序（运行samples\setup下示例程序所需要）。

------------------------------------------------------
setup : 安装卸载中间件所需要的dll和资源文件,示例请参考samples\setup目录。
